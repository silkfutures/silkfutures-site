# Silkfutures CIC — Website

Music, mentoring, and movement for young people in Cardiff.

## Structure

```
public/
  index.html    ← the full website (single file)
  images/       ← add your photos here
vercel.json     ← deployment config
```

## Adding images

1. Drop photos into `public/images/`
2. Reference them in `index.html` as `src="images/your-photo.jpg"`
3. Push to GitHub — Vercel auto-deploys

## Deployment

Connected to Vercel. Every push to `main` triggers a new deployment.
